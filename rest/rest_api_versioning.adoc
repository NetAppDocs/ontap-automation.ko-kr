= ONTAP REST API 버전 관리
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
ONTAP 9에 포함된 REST API에는 ONTAP 릴리즈 번호와 독립적인 버전 번호가 할당됩니다. ONTAP 릴리스에 포함된 API 버전과 API 사용에 어떤 영향을 줄 수 있는지 알고 있어야 합니다.


NOTE: ONTAP 9의 모든 버전은 동일한 버전의 REST API를 사용합니다.
